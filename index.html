tools4BCI - TiA Scope

Download this project as a .zip file Download this project as a tar.gz file

Why?

Especially for biosignal measurements, it is very often crucial to monitor recorded signals online. This has to be done to recognize potential artifacts soon enough. Otherwise, maybe the whole measurement would have to be repeated, e.g., like an EEG measurement contaminated by heavy EMG noise, caused by a participant who constantly clenched his face muscles. Thus, a simple scope application, utilizing the TiA protocol was implemented for this purpose.

Features

  • Usage of the TiA library with all its features
  • Cross platform
  • Continuous scrolling or cyclic update
  • Channel overlapping
  • Channel zooming
  • Variable plotting time window
  • Variable lowpass filter
  • Currently in development: Support for connection based channel selection and downsampling

Usage:

  • Zoom in/out all channels: Shift + mouse wheel
  • Zoom in/out single channel: Move mouse over desired channel; Ctrl + mouse wheel
